# ゲームブック書庫

選択肢で分岐する、ブラウザで読む物語。単一HTML・通信不要・インストール不要。

## ▶ 遊ぶ

**https://myzunota.github.io/gamebooks/**

上記URL(GitHub Pages)から作品一覧を開き、読みたい作品を選んでください。
スマートフォン/PCの両方に対応しています。

## 収録作品

| 作品 | 概要 |
|---|---|
| 残響の海 ― 軌道清掃船〈ヒバリ〉航海録 | 大破砕後の低軌道。デブリの海に漂う「死んだ機械たちの声」をめぐるSFファンタジー。結末6種。 |
| 沈まぬ者たち ― 泥海と白い都 ― 潜り屋遠征記 | 大地が泥にほどけた世界で、重さを民に移し替えて浮かぶ貴族に潜り屋が挑む。体力・所持品・予兆読み戦闘。結末6種+落命4種。 |

## 特徴

- 各作品は外部依存のない単一HTMLファイル。ダウンロードすればオフラインでも遊べます
- セーブはコード方式(書き出した文字列を控えて再開)。ブラウザには何も保存しません
- 「一手戻る」でいつでも選択をやり直せます。数字キーでの選択にも対応

## 構成

```
├─ index.html                          … 作品一覧(入口)
├─ zankyou_no_umi_gamebook.html        … 残響の海
└─ shizumanu_monotachi_gamebook.html   … 沈まぬ者たち
```

作品を追加する場合は、HTMLを1つ置いて `index.html` のカードを1枚増やすだけです。
